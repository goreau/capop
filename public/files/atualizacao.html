Materiais Auxiliares

  • Criado o histórico de atualizações para auxiliar na fase de implantação do sistema

Meu cadastro

  • permite edição de dados sem o preenchimento de todos os campos: altere o nome, login ou email, sem precisar preencher os campos de senha
  • para a alteração da senha, preencha os campos senha atual, nova senha e confirme a senha

Servidor

  • alterada a exclusão de servidor, seguindo uma das 3 possibilidades:
    1. Servidor não foi utilizado em nenhuma outra fonte de informação (uniforme, atividade de campo ou laboratório) -> o servidor é excluído permanentemente
    2. Servidor teve apenas seu uniforme inserido, sem outra fonte de informação -> a informação do uniforme e o servidor são excluídos permanentemente
    3. Servidor foi utilizados nas demais fontes de informação -> servidor é marcado como inativo e é informado ao usuário a fonte com dados relacionados.
      Dessa forma, o servidor não aparece mais nas listas de escolha de servidor em novas atividades, mas as informações onde foi utilizado ficam preservadas;
      Para excluir definitivamente o servidor, é preciso excluir ou editar os registros onde ele foi utilizado e depois repetir a rotina de exclusão;
      Quando não houver masi registros associados ao servidor, será seguida a rotina descrita no item 1
